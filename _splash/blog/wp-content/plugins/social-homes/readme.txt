== Installation ==

1. You need to have the Sidebar Widgets plugin installed.
1. You need to use a theme that supports Sidebar Widgets such as the WordPress 2.0 Default Theme (Kubrick), the Classic Theme, K2, ...
1. Put the 'socialhomes.php' file in 'wp-content/plugins/widgets/'.
1. Activate the Social Homes widget on the 'Plugins/Plugin Management' admin page.
1. Add the Social Homes widget to a sidebar on the 'Design/Widgets' admin page.
1. Configure the Social Homes services by clicking the 'Edit' button next to the Social Homes name on the 'Design/Widgets' admin page.
1. The order in which you configure the services is the order in which they will appear on the sidebar.
